# 少女前线脚本

## 注意

使用前要注意仓库容量，每次出发前需预留足够的床位

## 使用方法

在管理员权限下用cmd运行main.py文件，输入“x回车y”，“x”指计数器初始化的值，“y”指计数器终止拖尸的值，即y-x为一次拖尸次数

## 环境

- Python3.7.4 32-bit
- 模块
  - PyAutoGUI

## 功能

### 已实现

- 练级
  - 0-2 单打手
- 打捞
  - 1-6 1个AR
  - 2-6 1个AR
  - 2-4E 1个AR
  - 3-4E 二改M4+1个AR
  - 4-6 M4+射速HG
  - 4-4E 0-2拖尸队
  - 6-4E 顶配RF队

### 待实现

- 防错处理
