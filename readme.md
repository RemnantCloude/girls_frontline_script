## 注意

使用前要注意仓库容量，每次出发前需预留足够的床位

## 使用方法

在管理员权限下用cmd运行main.py文件，输入“x回车y”，“x”指计数器初始化的值，“y”指计数器终止拖尸的值，即y-x为一次拖尸次数

## 环境

- Python3.7.4 32-bit
- 模块
  - PyAutoGUI 0.9.46（**只支持32位Python**）
  - numpy 1.17.1
  - opencv-python 4.1.1.26
  - Pillow 6.1.0
